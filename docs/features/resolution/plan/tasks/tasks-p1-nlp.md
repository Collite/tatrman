# RG-P1 — `ttr-nlp` self-hosting (workstream C)

> Pre-flight: RG-P0 DONE (Q-10 sizing gates backend resource requests; Q-10 parity table = the S2 fixture seed; the fold lib exists). DoD: [`../plan.md`](../plan.md) §RG-P1. Contracts: [`../contracts.md`](../contracts.md) §1. Check each box the moment its task is done.
>
> **Shape of the work.** The front stays Python 3.13 / FastAPI + gRPC and becomes **engine-free** (contract + routing + langid; no torch, no models). Each model-bearing engine becomes its own backend image. The endpoint swap (Lindat → in-cluster) is *config* — the adapters already speak the protocol. **S-1 is a hard invariant:** no empty/default model anywhere; every response echoes engine + version.
