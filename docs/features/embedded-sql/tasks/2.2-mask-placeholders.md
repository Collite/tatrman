# 2.2 — `maskPlaceholders` span-preserving pre-pass (TDD)

**Repo:** modeler. **Effort:** ~0.5 day. **Package:** `@modeler/sql`.
**Pre-flight:** read contracts §3a and SPIKE S0.2 (the `{param}` finding — 167
occurrences, the entire raw-lex gap). The spike prototyped this in `src/mask.ts`;
this stage productionises it. **Required carrier component (E10)** — without it
T-SQL raw-lexes at ~41%.

---

- [ ] **2.2.1 — Spec first (red).** `src/__tests__/mask.test.ts`:
      - `maskPlaceholders('WHERE n = {nazev}')` → `masked === 'WHERE n = ·nazev·'`
        (braces → spaces), `placeholders === [{ offset: 10, length: 7, name: 'nazev' }]`
        (offset/length cover the **original** `{nazev}` span).
      - **Length invariant:** `masked.length === input.length` for every case.
      - Multiple placeholders; placeholder at string start/end; adjacent text.
      - Unbalanced `{` or stray `}` → left untouched (no replacement), no throw.
      - A `{` not followed by an identifier (`{ 1+1 }`) → left untouched.
      Red: `maskPlaceholders` doesn't exist yet.

- [ ] **2.2.2 — Implement.** `src/mask.ts` per contracts §3a: scan for
      `\{([A-Za-z_][A-Za-z0-9_]*)\}` (confirm the actual TTR placeholder grammar
      with the owner — match it exactly, not a guess); replace **only** the two
      brace chars with a single space each; push `{ offset, length, name }` using
      original-string offsets. Return `{ masked, placeholders }`. Pure function, no
      deps. ESLint-clean (no `any`).

- [ ] **2.2.3 — Property test the length invariant.** Add a small randomised test
      (or table of awkward inputs) asserting `masked.length === input.length`
      always — this is the contract that keeps the §8 source map uniform. A
      regression here silently corrupts every SQL span.

- [ ] **2.2.4 — Export from both subpaths.** `maskPlaceholders` must be in
      `@modeler/sql/lexers` (browser needs it before lexing) **and** `.`.

- [ ] **2.2.5 — Regression vs the spike numbers.** Add a test that masks a
      fixture containing the real `{param}` shapes from the S0.2 corpus and asserts
      zero residual `{`/`}` remain in `masked`. (This is the "restores 100% lex"
      guarantee at the unit level; the lexer-level proof is in 2.3.)

