# Tatrman (the standard & toolchain) — design pointer

> The standard's design record does **not** live here — it lives where it always has:
>
> - [`docs/features/`](../../../features/) — the per-effort design corpora (TTR-P, ttr-metadata, ttr-translator, grammar-master, md, semantics-block, qname-redesign, v1/v1-1/v2-1, …), each with its own control room and decision log.
> - [`docs/manual/`](../../../manual/) — the user-facing manual.
>
> This `ecosystem/tatrman/` folder holds only the **ecosystem-level** view: what the standard must deliver for the ecosystem's target versions — see [`../implementation/plan.md`](../implementation/plan.md).
>
> Ecosystem-level decisions affecting the toolchain (Apache-2.0, plan.v1 amendments, coordinates, naming) are recorded in the shared decision log: [`../../platform/design/00-control-room.md`](../../platform/design/00-control-room.md) §7.
