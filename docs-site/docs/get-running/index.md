# Get running

> **I have a database; show me the promise.**

This track takes you from a database you already own to a governed answer with
its provenance attached — in under an hour, using nothing but public artifacts
and these pages.

Start with the **[Quickstart](quickstart.md)**. It is the whole promise in seven
steps, and every step tells you what you should see before you move on.

## The pages

- **[Quickstart](quickstart.md)** — the seven-step path to your first governed
  answer, drafted against the published chart with real transcripts.
- **[Import a database](import-schema.md)** — the `ttr import-schema` reference:
  every flag, the conventions file, and what the importer writes.
- **[Review checklist](review-checklist.md)** — how to walk the import review,
  with a worked example.

Coming as the pilot onboards: a from-zero **kind** install, the per-engine
**connect-your-database** specifics (MSSQL, PostgreSQL), and a
**troubleshooting-first-contact** guide fed by real onboarding failures.
